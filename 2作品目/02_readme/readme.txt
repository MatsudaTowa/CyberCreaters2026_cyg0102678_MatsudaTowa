◆このゲームについて
刃がない剣士が柄のみで攻撃したり、特殊な斬撃を使い戦う横スクロールアクションゲーム

◆起動方法
フォルダ内のexeファイルを起動することでゲームスタート

◆基本操作
8/27時点:キーボードとマウスのみ

・キーボード
Aキー:左移動
Dキー:右移動
SPACEキー:ジャンプ(二段ジャンプ可能)
ENTERキー:画面遷移

・マウス
左クリック:斬撃発射(ゲージある時のみ、長押しでゲージの消費量が変わる)
右クリック:近接攻撃

・PAD
Lスティックor十字キー:移動
Aボタン:ジャンプ
Xボタン:斬撃発射(ゲージある時のみ、長押しでゲージの消費量が変わる)s
Bボタン:近接攻撃


◆実装内容
・プロトタイプ版時点(6/28)
プレイヤーとオブジェクトの当たり判定
プレイヤーの必須の動き(移動、ジャンプ、斬撃発射)
画面遷移
左右に動くだけの敵

・Reプロトタイプ版時点(7/4)
プレイヤーの近接攻撃追加
敵とオブジェクト、プレイヤー、プレイヤーの攻撃の当たり判定
ゲージを軽く実装

・Reプロトタイプ版時点(7/5)
マウスの入力の長さで斬撃のサイズが変わるように改良

・アルファ版時点(7/9)
敵の動きを改良、ノーマルエネミークラス追加

・アルファ版時点(7/19)
敵の攻撃を実装、プレイヤーの体力ゲージ追加
敵をすべて倒すか、自分が死ぬかでリザルト遷移

・アルファ版(欠席により遅れ)時点(8/01)
雑魚敵を倒すことでアイテムドロップ
それを拾うことで斬撃が使えるようになる
拾うまでは斬撃を打てないよう、ゲージもたまらないように変更

・ベータ版時点(8/27)
新しい敵と新しい斬撃追加
一定時間たつと連続斬撃を放ってくる(食らってる間スローにできたら尚よし)
倒してアイテムを取得すると範囲を出してその中にいる敵をロックオンしそこに連続斬撃を放つ
ダメージ時にスローを実装、デバッグ期間のレベルデザイン時に調整

・ベータ版時点(8/28)
ダメージ食らったときにマテリアル変更

・ベータ版時点(8/29)
